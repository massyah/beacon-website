    2 - Docker


    Docker is a tool designed to make it easier to create, deploy, and run applications by using containers. Developers use Docker to eliminate “works on my machine” problems when collaborating on code with co-workers.

    A Docker image of beacon is now available, this image contains :
  • the Elixir Beacon application deployed and running
  • a PostgreSQL database (with some sample data).
  • the JAR file with the default configuration

    A docker containing all components
    Database exported out of the container
