From sequencer to scientist

DNAstack is a managed, end-to-end solution for genomic data analysis and exploration.

All Demos

Upload

Upload large or small datasets through a simple and fast drag and drop uploader. FASTQ, BAM/CRAM, and VCF file file formats are supported for sequence, alignment, and variant call data, respectively.

Demo

Store

Datasets are stored in an infinitely scalable, secure cloud environment. We use industry standard encryption protocols for data at rest and in flight.

Process

Execute state-of-the-art bioinformatics pipelines, to convert raw sequence data into a list of genetic mutations from sequenced samples. DNAstack executes read alignment with BWA and variant calling with GATK.

Search through massive amounts of genetic variants using a high-performance faceted search engine which combines genotype, annotation, and patient characteristics.

Share

Selectively share genetic variant information through standard protocols developed by the Global Alliance for Genomics & Health, like Beacon. Easily subscribe datasets to be discoverable through the Beacon Network.

Develop

All features of DNAstack are available through a REST API. Bindings to common programmaing languages like Java make developing powerful applications and custom integrations simple.
